# Principles of Computer Design

__The Processor Performance Equation:__

```
CPU time = #Instructions * Cycles per Instruction * Clock Cycle Time

TExec = #I * CPI * TCC
```
__Amdahl Law__:

```
Speedup = OldExectime / NewExecTime = 1 / (f/s)
```

f - improved fraction with s
s - speedup
